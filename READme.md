## Building
#### Ubuntu 16.04 LTS
=====
- `sudo apt-get update && sudo apt-get upgrade --yes`
- `git clone https://github.com/NicNacDev/nicnaccoin-gui`
- `cd NicNacDev/cryptonote/`
- `git clone https://github.com/NicNacCoins/NicNacCoin . `
- `cd ..`
- `mkdir build && cd $_`
- `cmake ..`
- `make -j4`
